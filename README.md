# Matterwiki for Java




According to the reactjs project [Matterwiki](https://github.com/Matterwiki/Matterwiki), 
I rewrite in Java  base on springboot. 




Just run this command  to start it


```
java -jar  matterwiki-4j-boot-1.0-SNAPSHOT.jar
```
